# GEMINI Tech Stack Analysis

This document outlines the technology stack of the current project.

## Backend

- **Language:** Java 8
- **Framework:** Spring Boot 2.7.18
- **Web Server:** Embedded Tomcat (from `spring-boot-starter-web`)

## Frontend

- **markup:** HTML

## Build Tool

- **Tool:** Gradle

## Key Dependencies

- `spring-boot-starter-web`: Core dependency for building web applications with Spring Boot.
- `logback-classic`: A robust logging framework for Java.
- `lombok`: A library to reduce boilerplate code for model/data objects.
- `spring-boot-devtools`: Provides development-time features like automatic restarts and live reload.
